# Documentación del proyecto

## fall-detection-system

Este directorio contiene la documentación técnica y funcional del proyecto **fall-detection-system**, desarrollado en el Ciclo Formativo de Grado Superior en Desarrollo de Aplicaciones Multiplataforma (DAM), dentro de la asignatura **Desarrollo de Interfaces**.

El sistema implementa una solución IoT para la detección automática de caídas en personas mayores, combinando hardware, procesamiento en tiempo real y una arquitectura web full-stack.

---

## Contenido de la documentación

1. Arquitectura del sistema y flujo de detección de caídas.
2. Modelo de datos y diseño de bases de datos.
3. Manual de usuario.
4. Manual técnico.
5. Reparto de tareas del equipo.

## Objetivo

La documentación ha sido elaborada siguiendo criterios profesionales, con el objetivo de:

- Justificar las decisiones tecnológicas adoptadas.
